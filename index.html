How the estimate works

  1. If entered, an item discount is removed from the shelf price first.
  2. If enabled, the Annual Pass discount is then applied on top.
  3. The app extracts the VAT portion from that discounted VAT-inclusive price.
  4. If enabled, it applies the configured SkipTax refund share to that VAT amount.
  5. The remaining Euro total is converted to GBP with the latest saved exchange rate.

This is meant as a quick park-side estimate rather than a bank-perfect conversion.
